# TDAH Ideias 💡

App para coletar ideias ao longo da semana e organizá-las com IA.  
Feito para quem tem TDAH e esquece de voltar nas anotações.

---

## Como instalar no celular (GitHub Pages)

### Passo 1 — Criar conta no GitHub
Acesse [github.com](https://github.com) e crie uma conta gratuita.

### Passo 2 — Criar repositório
1. Clique em **New** (botão verde no canto superior esquerdo)
2. Nome do repositório: `tdah-ideias`
3. Marque **Public**
4. Clique em **Create repository**

### Passo 3 — Subir os arquivos
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste todos os arquivos desta pasta para lá:
   - `index.html`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`
3. Clique em **Commit changes**

### Passo 4 — Ativar GitHub Pages
1. Vá em **Settings** (aba no topo do repositório)
2. No menu lateral, clique em **Pages**
3. Em "Source", selecione **Deploy from a branch**
4. Branch: **main** / pasta: **/ (root)**
5. Clique em **Save**
6. Aguarde 1-2 minutos — o link aparecerá no topo da página

Seu app estará em: `https://seuusuario.github.io/tdah-ideias`

### Passo 5 — Instalar no Android
1. Abra o link no **Chrome**
2. Toque nos três pontinhos (⋮) no canto superior direito
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — o ícone aparece na home do celular ✓

---

## Como usar

1. **Digite a ideia** no campo de texto — qualquer coisa, bruta mesmo
2. Toque em **Salvar ideia** (ou Ctrl+Enter no computador)
3. Vai acumulando ao longo da semana
4. Quando quiser, toque em **Organizar tudo com IA**
5. A IA agrupa por tema, formata e devolve organizado
6. Copie o resultado e mande pra você mesmo no WhatsApp

---

## Configurar a chave de IA

Na primeira vez que usar o botão "Organizar", o app vai pedir sua chave da API da Anthropic.

1. Acesse [console.anthropic.com](https://console.anthropic.com)
2. Crie uma conta — tem crédito gratuito pra começar
3. Vá em **API Keys** e crie uma chave
4. Cole no app (⚙ no canto superior direito)

A chave fica salva só no seu celular. Nunca sai do dispositivo.
